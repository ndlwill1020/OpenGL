视图矩阵V：定义相机空间，相机位于相机空间的原点，用一个矩阵表示相机空间与世界坐标系之间的变换关系
视图矩阵：将顶点从世界坐标系转换到相机坐标系（对顶点操作）
姿态矩阵：将相机当作一个普通物体，进行空间变换（对相机操作），同三维变换

视图矩阵和姿态矩阵互为逆矩阵＃＃＃＃＃＃
－－－－－－－－－－－－－－－－－－－－相机的平移：
将相机从原点平移到（x,y,z）
视图矩阵为：
1 0 0 -x
0 1 0 -y
0 0 1 -z
0 0 0 1

－－－－－－－－－－－－－－－－－－－－相机的旋转：
相机空间：UVN相机
N：这个向量表示相机的朝向，称为“look at”，与z轴相关联
V：这个向量是头顶指向天空的方向（upright），与y轴相关联
U：这个向量指向右侧（right side），与x轴相关联

Ux Uy Uz 0   x_world     x_camera
Vx Vy Vz 0 * y_world  =  y_camera
Nx Ny Nz 0   z_world     z_camera
0 0 0 1       1          1
